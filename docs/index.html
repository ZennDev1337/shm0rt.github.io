The Manual: How to Work with Me

Working Style

  • Tasks run best with direct, clear requests. Extra fluff slows everything down.
  • Providing full context at the start ensures accurate execution. Once direction is clear, short requests are fine.
  • Daily check-ins help keep alignment. For urgent needs, calling is fastest.

Availability

  • A structured schedule is preferred, though overtime is possible if the project is engaging.
  • Messages are read carefully. Replies may not be instant, but each concern is addressed thoughtfully.
  • Avoid excessive messaging; it disrupts focus.

How to Get the Best Results

  • Clear instructions. The more detail, the better the output.
  • Minimalism is key—no bloat. Use tools only if they solve real problems.
  • Trust the process. Focus on end goals, not micromanagement.

Things to Avoid

  • Unstructured or vague objectives slow down progress.
  • Last-minute changes only if they genuinely improve outcomes.
  • Over-management can hinder efficiency.

Contact

  • Responses come as soon as possible, factoring in workload.
